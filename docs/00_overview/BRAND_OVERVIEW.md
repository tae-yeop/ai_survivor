# Brand Overview

Status: Active
Last Updated: 2026-05-09

## 브랜드명

- 한국어 표시명: **AI 시대 생존기**
- 영문/slug 성격 이름: **AI Survivor**
- 운영 도메인: 미확정. 도메인 구매 전에는 실제 Vercel production URL을 사용하고, 구매 후 운영 도메인으로 교체한다.

브랜드는 “AI가 잘한다고 말하는 시대에, 직접 도구를 써보고 살아남는 기록”을 지향한다.

## 사용 문장

**AI 시대 생존기는 AI 도구와 튜토리얼을 직접 따라 해보고, 진짜 되는지 검증하는 실전 생존 블로그다.**

중심 문장:

> AI 튜토리얼, 제가 먼저 끝까지 해봅니다.

## 톤

- 친근하지만 정확하게 쓴다.
- 과장보다 실험 결과를 앞세운다.
- 성공만 쓰지 않고 막힌 부분과 비용도 같이 쓴다.
- 독자가 복붙해서 시작할 수 있는 설정, 프롬프트, 코드, 체크리스트를 남긴다.

## 사이트 카피 기준

### Hero

- AI 튜토리얼, 제가 먼저 끝까지 해봅니다.
- 설치부터 에러, 비용, 결과물까지, 살아남은 것만 정리합니다.

### About

AI 시대 생존기는 새 도구 소식을 빠르게 전달하는 블로그가 아니다. 직접 써보고 막히고 비용을 확인하고 결과물을 만든 뒤, 남길 가치가 있는 것만 기록한다.

### Footer

직접 해본 AI 튜토리얼과 도구 사용기, 되는 것만 남깁니다.

## 콘텐츠 확장

블로그 글 하나는 다음 채널로 쪼갤 수 있어야 한다.

- 인스타 카드뉴스: 문제, 준비물, 막힌 부분, 해결, 비용, 결론
- 유튜브: 실제 따라 하기 화면과 결과물 중심
- Shorts/Reels: 세팅 하나, 에러 하나, “막혔다가 해결” 같은 짧은 생존 팁
