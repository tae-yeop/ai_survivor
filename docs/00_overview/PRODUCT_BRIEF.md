# Product Brief

## 1. 한 줄 설명

**AI 시대 생존기**는 AI 도구와 튜토리얼을 직접 끝까지 따라 해보고, 진짜 되는 방법만 남기는 실전 생존 블로그다.

뉴스 요약 블로그가 아니다. 도구 소개 블로그도 아니다. 설치, 에러, 비용, 결과물, 한계까지 사람이 직접 겪은 기록을 남긴다.

## 2. 제품 약속

독자는 글 하나를 읽고 다음 중 하나를 얻어야 한다.

- 같은 도구를 바로 설치하고 써볼 수 있다.
- 막히는 지점을 미리 알고 피해 갈 수 있다.
- 비용, 품질, 속도를 대략 예상할 수 있다.
- 따라 할지 말지 판단할 수 있다.
- 결과물을 업무, 개발, 콘텐츠 제작, 서비스 제작에 연결할 수 있다.

## 3. 핵심 독자

### Primary

- AI를 실제 업무에 붙이고 싶은 개발자, 직장인, 1인 메이커
- Claude Code, Cursor, ChatGPT, 음성/영상 AI, 자동화 도구를 써보고 싶은 사람
- 튜토리얼을 봐도 중간에 막혀서 끝까지 못 간 경험이 있는 사람

### Secondary

- AI 도구 사용기를 콘텐츠나 강의로 재가공하려는 사람
- 검색으로 들어와 특정 에러, 설정, 비용 정보를 찾는 사람
- 블로그 이후 인스타, 유튜브, 쇼츠로 확장될 짧은 실전 팁을 찾는 사람

## 4. 차별점

| 흔한 AI 콘텐츠 | AI 시대 생존기 기준 |
| --- | --- |
| 새 도구 소개 | 직접 써보고 끝까지 간 결과 |
| 기능 요약 | 설치, 에러, 비용, 결과물 |
| 홍보성 장점 | 추천/비추천 대상 분리 |
| 캡처 없는 설명 | 프롬프트, 코드, 설정, 산출물 포함 |
| 뉴스형 속보 | 지금도 되는지 확인한 기록 |

## 5. MVP 범위

### 포함

- MDX 기반 게시글
- 카테고리/태그/시리즈 탐색
- GitHub backed admin editor
- 로그인 후 브라우저 내 직접 편집
- draft/published/scheduled/archived 상태
- RSS, sitemap, robots.txt
- About, Contact, Privacy
- Vercel 배포

### 제외

- 공개 회원가입
- 공개 댓글
- 결제/구독
- 외부 CMS 도입
- DB 기반 검색
- 완전한 미디어 DAM

## 6. 수익화 방향

### Phase 1

검색 유입을 받을 수 있는 실전 튜토리얼과 도구 사용기 축적.

### Phase 2

AdSense, 제휴 링크, 도구별 허브 페이지 검토.

### Phase 3+

인스타 카드뉴스, 유튜브 튜토리얼, 쇼츠/Reels로 같은 실험을 재가공.

## 7. 성공 기준

- 글이 검색으로 발견된다.
- 독자가 글을 보고 같은 결과물을 재현한다.
- 실패한 부분도 정보로 남아 신뢰를 만든다.
- 운영자가 브라우저에서 글을 빠르게 고치고 publish 할 수 있다.
- 콘텐츠가 블로그, 인스타, 유튜브로 자연스럽게 확장된다.

## 8. 비원칙

- 직접 해보지 않은 도구를 추천하지 않는다.
- 비용과 실패를 숨기지 않는다.
- LLM이 만든 일반론을 그대로 게시하지 않는다.
- 뉴스보다 검증을 우선한다.
